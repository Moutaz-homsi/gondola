# Getting Started with the backend
`cd backend`
`cp .env.example .env`
`yarn install`

## For development env
`yarn develop`

## For production env
Edit .env as you want and then =>
`yarn build`
`yarn start`

# Getting Started with the frontend
`cd frontend`
`cp .env.example .env`
`yarn install`

## For development env
`yarn start`

## For production env
Edit .env as you want and then =>
`yarn build`
`yarn dlx serve -s build`
